# ScholarX SaaS Billing Portal

A role-based SaaS Billing Portal built using React and Node.js.

## Features

- User Dashboard
- Billing History
- Subscription Plans
- Admin Panel
- Role-Based Access
- Responsive User Interface

## Tech Stack

- Frontend: React.js
- Backend: Node.js
- Server: Express.js
- Styling: CSS

## Features Implemented

### User Dashboard
- Displays current subscription plan.
- Shows billing history.

### Subscription Plans
- Basic Plan
- Premium Plan

### Admin Panel
- View users and roles.
- Role-based access control.

## Installation

### Frontend

```bash
cd client
npm install
npm start
```

### Backend

```bash
cd server
npm install
node server.js
```
